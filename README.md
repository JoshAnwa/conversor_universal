# Conversor Universal de Unidades

## Descripción del Proyecto

Este es un programa de consola desarrollado en Python que sirve como un conversor universal de unidades. El objetivo es ofrecer una herramienta simple y funcional que aplique buenas prácticas de programación (uso de funciones modulares) para realizar cálculos de conversión de manera eficiente.

El proyecto cumple con los requisitos de la Parte 1, incluyendo una interfaz de texto, entrada de datos, lógica de procesamiento y salida clara.

### Unidades Soportadas (10 Conversiones)

El conversor está organizado en cuatro categorías principales:

| Categoría | Conversiones |
| :--- | :--- |
| **Temperatura** | Celsius a Fahrenheit, Celsius a Kelvin, Fahrenheit a Celsius. |
| **Distancia** | Kilómetros a Millas, Metros a Pies, Pies a Centímetros. |
| **Masa** | Kilogramos a Libras, Gramos a Onzas. |
| **Almacenamiento Digital** | Megabytes a Gigabytes, Gigabytes a Terabytes. |

## Cómo se Instala

El proyecto es ligero y no necesita librerías externas ni gestores de paquetes.

### Requisitos

* **Python 3.x** (Versión utilizada para el desarrollo: [VERSION DE PYTHON QUE USARON, ej: Python 3.11])

### Pasos de Ejecución

1.  **Obtener el Código:**
    * *Si ya descargaste el zip:* Descomprime la carpeta del proyecto.
    * *Si clonaste el repositorio (Opción preferida para la Parte 2):*
        ```bash
        git clone (https://github.com/JoshAnwa/conversor_universal)]
        cd Conversor
        ```
2.  **Ejecutar el programa:**
    Abre tu terminal (o Símbolo del Sistema) en la carpeta raíz del proyecto y ejecuta el archivo principal:
    ```bash
    python conversor.py
    ```

## Cómo se Usa

El programa funciona completamente a través de la interfaz de línea de comandos:

1.  Al iniciar, el programa presenta un menú con las 10 opciones de conversión (del 1 al 10) y la opción 11 para salir.
2.  **Ingresa el número** de la conversión que deseas realizar (ej. `4` para Kilómetros a Millas).
3.  El programa te solicitará **el valor numérico** que deseas convertir (ej. `100`).
4.  El resultado se muestra en pantalla con la unidad de medida correspondiente.
